# How To Share The TPC Animation

Use this file:

`copy_paste_tpc_animation.html`

It is a single standalone HTML file. It already contains the CSS, SVG diagrams, and JavaScript animation controls. It does not need any extra images, libraries, or folders.

## Easiest Options

### Option 1: Upload the HTML file somewhere

Upload `copy_paste_tpc_animation.html` to any static website host.

Good simple choices:

- GitHub Pages
- Netlify Drop
- Vercel
- Neocities

For Netlify Drop, drag the HTML file onto:

https://app.netlify.com/drop

### Option 2: Paste into an online HTML editor

Open one of these:

- https://codepen.io/pen/
- https://jsfiddle.net/
- https://playcode.io/html

Then paste the full contents of `copy_paste_tpc_animation.html`.

If the editor has separate HTML, CSS, and JS panels, use the full-page or complete-document mode if available. Otherwise, it is easier to upload the file directly instead of splitting it apart.

### Option 3: Send the file directly

Send `copy_paste_tpc_animation.html` to someone. They can double-click it and it should open in their browser.

## Important GitHub Note

Clicking `index.html` inside the GitHub repository shows the code. That is normal.

To see the animation on GitHub, the repo needs GitHub Pages enabled, and people should open the GitHub Pages website link, not the code-file link.
